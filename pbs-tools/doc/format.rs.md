# File Analysis: `format.rs`

## Purpose
Utilities for formatting data types into human-readable strings.

## Features
* **Byte Sizes**: formatting numbers as `B`, `KiB`, `MiB`, `GiB`, `TiB`.
* **Hex Dump**: Formatting binary data as hex strings for debugging.
* **Durations**: Formatting `std::time::Duration` into "2h 30m" style strings.
